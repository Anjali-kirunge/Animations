# 🎨 CSS Animations Project

This project demonstrates various creative animations using pure CSS and HTML. It includes animated shapes, images, and effects to showcase front-end design skills.

## 🚀 Features
- Animated image containers
- Text animation (ARTIST effect)
- Rotating and moving shapes
- Triangle and circle animations
- Analog clock animation ⏰
- Butterfly image animation
- Use of Animate.css library

## 🛠️ Tech Stack
- HTML5
- CSS3 (Animations & Transitions)
- Animate.css

## 📂 Project Structure
- `index.html` – Main HTML file
- `style.css` – Contains all animation styles
- Images – Used for animation effects

## 🎯 Key Highlights
- Creative use of CSS keyframes
- Multiple animation layouts
- Visual effects without JavaScript
- Beginner-friendly animation concepts

## 📌 How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
